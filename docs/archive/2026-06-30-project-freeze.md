# プロジェクト凍結メモ

**凍結日**: 2026-06-30  
**理由**: JR東日本スタートアッププログラム 不採択

---

## 背景

JR東日本スタートアッププログラムへの応募に向けて、改札内施設の購買前行動可視化・送客最適化をテーマに PoC を開発・提案した。
応募書類・Webデモ・提案資料を整備した上で応募したが、不採択の結果となった。

不採択を受けて、すぐに他の私鉄や交通事業者向けに作り替える方向にはいかず、いったん凍結する。
理由は以下の通り。

- 提案テーマ自体（購買前行動データ基盤）の価値は変わらない
- 鉄道会社・駅ビル会社との接点なしに作り替えても、PoC が宙に浮く
- コードと設計資産は十分に再利用可能な状態にあり、急いで解体する必要はない
- 次の接点が出た時に、この資産をベースに再提案する方が現実的

---

## 資産の評価

このプロジェクトの価値は「地図アプリ」「施設検索アプリ」ではなく、以下の事業構想にある。

| 資産の本質 | 説明 |
|---|---|
| **駅空間DX** | 改札内施設データを起点に、駅ナカ商業の運営・送客・情報更新をデジタル化する構想 |
| **駅ナカ送客・リテールメディア** | 目的別導線・チェックイン・UGC更新により、個店へのデジタル送客を実現するプラットフォーム |
| **購買前行動データ基盤** | POSに残らない探索・滞在・離脱・接触の利用ログを取得・分析・活用する仕組み |
| **公共交通インフラ × 商業施設PoC** | 改札内という物理的なクローズド空間でのデータ基盤実証として横展開可能 |

コード・施設データ・設計ドキュメント・提案資料はいずれも再利用可能な状態で保存する。

## 再開の方向性

再開する場合は、以下のような接点・状況が生まれた時を想定する。

- 鉄道会社（私鉄・JR各社）や駅ビル会社との商談・公募への参加機会
- 交通広告会社・商業施設会社からの購買前行動データ活用に関する打診
- リテールメディア・駅ナカDXを主テーマとした公募・補助金への応募機会

詳細は [restart-conditions.md](./restart-conditions.md) を参照。
