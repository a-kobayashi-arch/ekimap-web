# 08 新セッション開始指示

> このファイルの「## 開始指示」以下を、新しい Claude Code セッションの最初のメッセージとして貼り付ける。

---

## 開始指示

```
まず以下の2ファイルを読んでください。

1. CLAUDE.md
2. docs/07_session-handover.md

読み終わったら、**実装はまだ始めず**、以下を報告してください。

### 1. プロジェクト目的の要約
（3行以内で）

### 2. 現フェーズ
JR東日本スタートアップ応募前フェーズかどうか、
次に実装すべき最小改修は何か。

### 3. 触ってはいけない領域
（箇条書きで列挙）

### 4. 次に触る予定のファイル
（具体的なパスで列挙）

### 5. リスクがある箇所
既存機能・KVキー・型定義・APIに対して影響が出うる箇所を列挙。

---

報告を確認後、こちらから「進めてください」と指示します。
それまで実装・ファイル編集・git操作は行わないでください。
```

---

## このファイルの使い方

1. 新しい Claude Code セッションを開始する
2. 上の「## 開始指示」内のコードブロックをそのままコピーして貼り付ける
3. Claude が報告を返したら内容を確認する
4. 問題なければ「進めてください」と返信する

## 注意

- 開始指示を貼る前に、前セッションで `docs/07_session-handover.md` が更新されていることを確認する
- `git log --oneline -3` と `git status` で状態を確認してから新セッションを始めることを推奨
