# 05 JR東日本スタートアップ応募 PoC

## 応募の目的・背景

JR東日本スタートアップ向けに、本アプリを「改札内回遊・滞在価値・店舗送客を可視化する駅ナカ運営 DX/販促基盤」として提案する。

一般ユーザー向けアプリをそのまま見せるのではなく、**B2B 文脈で JR の課題に答える見せ方に再編集**することが目的。

---

## PoC テーマ

> **「改札内で今すぐ座れて充電できる/立ち寄れる場所の可視化と送客最適化」**

## デザイン方針

### 一般向けページ

- 探索感・楽しさ・親しみやすさを優先
- スタンプラリー・ゲーム性のある演出

### JR 向けページ

- 落ち着いた B2B トーン
- シンプルで整理された情報設計
- 過度な装飾・演出は不要
- **資料に貼れる見た目を優先**（スクリーンショット撮影・応募資料転用）

---

## JR 向けページで前面に出さないもの

- スタンプラリー・路線制覇感
- ゲーム性の強い演出
- 全国展開の夢
- 未実装の大規模機能

## 目的別フィルタの実装（`jr/demo/page.tsx`）

```typescript
const PURPOSE_FILTERS = [
  { id: "eat",    label: "🍽️ 食べる", match: f => f.category === "飲食店" },
  { id: "buy",    label: "🛍️ 買う",   match: f => ["ショップ","食材・お土産","雑貨・文具"].includes(f.category) },
  { id: "sit",    label: "🪑 座る",   match: f => f.seating === "yes" },
  { id: "charge", label: "⚡ 充電",   match: f => f.outlet === "available" },
];
```

**「充電」は `outlet: "available"` のみ。`unknown` は含まない。**

---

## 充電タブの現状と対応

現在、JR デモ主力 3 駅（新宿・大宮・赤羽）はいずれも `outlet: "available"` = 0 件。  
充電タブを選択すると専用の空表示文言が出る（v0.15.2 対応済み）:

> 「現在、客用コンセントが確認できている施設はありません。現地確認・投稿により順次更新されます。」

**現地調査で客用コンセントを確認した施設があれば、`outlet: "available"` に設定して充電タブを有効化する。**
